    가위바위보 프로젝트

    간단한 가위바위보 프로그램입니다. 현재는 컴퓨터와 겨루게 되어있습니다.

    $ git clone git://github.com/kenu/gawi.git

    예제 페이지 : okjsp gawi

    JSON request

    데이터는 Ajax로 전달합니다. API는 다음과 같습니다.

    URL: http://www.okjsp.pe.kr/gawi/statJSON.jsp
request: {}
response: 
{"total":12789, "win":4274, "even":4193, "lose": 4322, "rate": "33.41%"}
    URL: http://www.okjsp.pe.kr/gawi/queryJSON.jsp?choice=0
request: ?choice=0
0: 가위
1: 바위
2: 보
response: 
{
"p1":{"name":"당신", "choice":"바위"}, 
"p2":{"name":"컴퓨터", "choice":"바위"}, 
"judgement" : "비겼습니다.", 
"stat" : {"total":12790, "win":4274, "even":4194, "lose": 4322, "rate": "33.41%"} 
}
